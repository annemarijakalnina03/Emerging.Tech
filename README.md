# Aircraft Surface Defect Detection Using AI

## 📌 Project Description

This project aims to develop an image recognition system to detect defects on aircraft surfaces using machine learning techniques. The system classifies images into damaged and undamaged categories.

## 🎯 Objectives

* Automate aircraft surface inspection
* Reduce human error
* Improve maintenance efficiency

## 🧠 Methodology

* Dataset collected from online sources (Kaggle)
* Images preprocessed (resized, normalized)
* CNN model used for classification
* Model trained and evaluated

## 📊 Results

* Model achieved ~85–95% accuracy 
* Successfully classified defects in test images
